class Account(*args, **kwargs)¶

Bases: django.db.models.base.Model

Represents a row in the Account database.

Note to teammate: Maps one-to-one with Django’s build-in User object User object has username, password, first_name and last_name fields, takes care of password encryption.

account.id –> id in db account.user –> associated User object account.user.username –> username account.user.password –> password account.user.first_name –> first name of user account.user.last_name –> last name of user account.authority –> authority of user (currently 1,2,3)
